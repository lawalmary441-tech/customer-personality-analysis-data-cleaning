# Customer Personality Analysis Data Cleaning

## Project Overview
This project focuses on cleaning and preprocessing a messy Customer Personality Analysis dataset using Microsoft Excel.

## Dataset
The dataset contains customer demographic and purchasing information, including:
- Customer ID
- Year of Birth
- Education
- Marital Status
- Income
- Kid Home
- Teen Home
- Customer Registration Date
- Amount Spent on Products
- Number of Purchases
- Website Visits
- Customer Rating
- Country

## Data Cleaning Tasks
- Removed duplicate records
- Handled missing values
- Corrected inconsistent text values
- Standardized date formats
- Replaced invalid values
- Prepared the dataset for analysis

- Raw Dataset
- Cleaned Dataset
